# Contributing to helplib

First off, thanks for taking the time to contribute! 🎉

When contributing to this repository, please first discuss the change you wish to make via issue, email, or any other method with the owners of this repository before making a change.

## Pull Request Process

1. Fork the repository and create your branch from `main`.
2. Install dependencies by running `npm install`.
3. Make your changes and ensure your code is strictly typed with TypeScript.
4. If you've added code that should be tested, add tests.
5. Ensure the test suite passes and the project builds successfully by running `npm run build`.
6. Update the README.md with details of changes to the interface, if applicable.
7. Submit your Pull Request!

## Reporting Bugs

If you find a bug in the source code, you can help us by submitting an issue to our GitHub Repository. Even better, you can submit a Pull Request with a fix.
